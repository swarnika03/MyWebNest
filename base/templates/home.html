Generative AI for Drug Discovery

Tools: Python, NumPy, Pandas, RDKit, Scikit-learn, Random Forest, KNN, Decision Tree
Dataset: pCHEMBL240 (ChEMBL bioactivity data)

Developed machine learning models to predict drug-target bioactivity using the pCHEMBL240 dataset. Extracted molecular descriptors using RDKit and applied various regression algorithms, including Random Forest, KNN, and Decision Trees. The Random Forest Regressor achieved the best results with R² = 0.9526 and MAE = 0.3218. Conducted performance evaluation using MAE, MSE, RMSE, and R². Documented findings in a research-style report and highlighted the potential for using generative AI techniques in future drug discovery pipelines.
